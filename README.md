# Xtrack
![Xtrack preview](Xtrack.jpg)

## introduction
Xtrack is a tracking tool that can be used to track IP addresses, telephone numbers and usernames.

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ apt update -y && apt upgrade -y
$ pkg install git
$ pkg install python -y
$ git clone https://github.com/Whomrx666/Xtrack.git
$ cd Xtrack
$ pip install -r requirements
$ python Xtrack.py
```

## Instructions
- **first**: Install tools according to the instructions above
- **second**: Select one of the selected columns that you want to track
- **third**: Enter the IP address, telephone number, or username according to the column you chose
- **last**: The tools will automatically track according to your commands
